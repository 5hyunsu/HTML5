FORM태그내에SUBMIT 버튼 클릭시 해당 FORM안의 작성된
사용자가 입력한 값들을 서버로 전송하면서 요청하는 역할 수행

FORM ACTION =""METHOD=""

* action 속성: 해당 form에 사용자가 입력한 값을 전달하면서 요청할 서버의 경로제시 표시 * method 속성: 요청 전송 방식을 지정하는 속성 (get(기본값 ) | post)
> get 방식 : 요청시 사용자가 입력한 값 들이 url에 노출되는 방식
> post 방식 : 요청시 사용자가 입력한 값 들이 url에 노출되지 않는 방식
> 스프링 할 때는 또 다른 방법
form 내의 submit 버튼 클릭시 action지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청됨(사용자가 넣은 값은 key=value 세트로 전달)
